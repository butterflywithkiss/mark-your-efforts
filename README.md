Mark Your Efforts

A personal productivity and consistency tracking web application designed to help users record daily efforts, monitor progress, and build long-term discipline.

This project was built with the idea that **small consistent actions create big results over time**.

Purpose

Many people struggle with staying consistent in their goals.  
This application helps users:

- Track daily efforts and habits
- Stay motivated through visible progress
- Build long-term discipline
- Reflect on personal growth

The main focus is **consistency over perfection**.

 Features

-  Add and manage daily tasks or efforts
-  Track progress over time
-  Save user entries locally or in database (depending on setup)
-  View past activity and consistency
-  Simple and clean user experience
